## 📖 Project Description

This project classifies Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

using the K-Nearest Neighbors (KNN) Machine Learning algorithm.

---

## 🛠 Technologies Used

- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

---

## 📂 Workflow

1. Load Iris Dataset
2. Data Exploration
3. Data Visualization
4. Train-Test Split
5. Feature Scaling
6. Train KNN Model
7. Prediction
8. Accuracy Evaluation
9. Confusion Matrix
10. Classification Report

---

## 📈 Result

Model Accuracy: **96%–100%**

The model successfully predicts the flower species using four flower measurements.
